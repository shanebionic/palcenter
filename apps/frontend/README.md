# PalCenter frontend

This is the Next.js and Mantine web interface for PalCenter. It is built and
deployed together with the Fastify API; it is not a standalone Vercel
application.

Run commands from the repository root:

```sh
pnpm dev
pnpm check-types
pnpm lint
pnpm build
```

See the repository [README](../../README.md) for deployment, security, backup,
upgrade, and release instructions.
